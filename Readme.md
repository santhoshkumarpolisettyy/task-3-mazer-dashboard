## 📊 Data Integration

The dashboard dynamically displays data using a local `data.json` file.

### Example:

```json
{
  "views": 999,
  "followers": 1500,
  "following": 350,
  "posts": 75,
  "tasks": 25
}
```

JavaScript fetch API is used to load and display this data in the UI.

---

## 🧠 How It Works

1. Data is stored in `data.json`
2. JavaScript fetches the data
3. DOM elements are updated dynamically
4. UI reflects real-time values
