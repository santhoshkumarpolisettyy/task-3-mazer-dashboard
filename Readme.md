# Task 3 - Mazer Dashboard Customization

## 📌 Overview
This project is a customized version of the Mazer Admin Dashboard template. The goal is to demonstrate front-end skills including UI customization and data integration.

## 🎯 Features
- Customized dashboard UI
- Added personal details and styling
- Integrated JSON data dynamically using JavaScript
- Responsive design using Bootstrap 5

## 🛠️ Technologies Used
- HTML
- CSS (Bootstrap 5)
- JavaScript
- JSON

## 📊 Data Integration
A `data.json` file is used to store values like:
- Profile Views
- Followers
- Following
- Posts
- Tasks

JavaScript fetch API is used to dynamically update UI elements.

## 🚀 How to Run
1. Open project folder
2. Run `index.html` using Live Server
3. Dashboard will load with dynamic data

## 📁 Project Structure
- `index.html` → Main UI
- `data.json` → Data source
- `script.js` → Fetch and update logic
- `assets/` → CSS, JS, images

## ✅ Outcome
A fully functional, responsive dashboard with dynamic data integration.
